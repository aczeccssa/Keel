Architecture & Framework Components

Component Feature Implementation Class Technical Description
Plugin Isolation PluginTable(pluginId, name) Auto-prefixes table name with pluginId for strict schema isolation. Physical mapping: dbdemo_notes
Audit Trailing AuditPluginTable Composites TimestampTable + SoftDeletableTable with structured createdBy/updatedBy fields.
Temporal Management TimestampTable Automated created_at and updated_at synchronization via kotlinx.datetime.Instant.
Soft Delete Paradigm SoftDeletableTable Enforces logical deletion via nullable deleted_at column; active operational records retain null.
CRUD Abstraction CrudRepository<ID, E> Standardized interface for findById, findAll, save, deleteById, existsById, and count operations.
Transaction Context BaseRepository<T> Lifecycle wrappers: transaction{}, transactionWithResult{}, encapsulating robust logError() handling.
Connection Pooling DatabaseFactory.h2Memory() HikariCP integration + Exposed ORM layer. Agnostic support for H2, PostgreSQL, MySQL, and SQLite.
Database Lifecycle KeelDatabase Manages DDL execution (create/drop) and robust asynchronous wrappers (suspendTransaction, transactionWithRetry).
